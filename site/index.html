Как запустить

  1. 01Клонируйте репозиторий или скачайте архив с GitHub.
  2. 02Создайте окружение: python -m venv .venv.
  3. 03Поставьте зависимости: pip install -r requirements.txt.
  4. 04Запустите сервис: python app.py.

GitHub Pages показывает эту страницу. Сам транскрибатор запускается локально, потому что ему нужны Python, обработка файлов и приватный API-ключ.
